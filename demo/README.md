A festive Christmas countdown timer built with PyScript, HTML, and CSS. This web application displays the time remaining until Christmas, accompanied by a beautiful animation of falling snowflakes.

## Features

- **Real-time Countdown:** Shows the remaining days, hours, minutes, and seconds until Christmas.
- **Festive Animations:** Includes an animation of falling snowflakes to create a wintery atmosphere.
- **Modern Design:** A clean and beautiful design with a dark blue gradient background and a custom festive font.
- **Built with Python:** The entire logic for the countdown and animations is written in Python, thanks to the power of PyScript.

## Technologies Used

- [PyScript](https://pyscript.com/): A platform for running Python in the browser.
- HTML5
- CSS3

## How to Run

1.  **Start a local web server:**
    - Make sure you have Python installed on your system.
    - Open your terminal and navigate to the `demo` directory.
    - Run the following command:
      ```bash
      python -m http.server
      ```
2.  **Open in your browser:**
    - Open your web browser and go to `http://localhost:8000`.
